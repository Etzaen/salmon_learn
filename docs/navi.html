サーモンラン関連で作ったものまとめ by ガンジー（@GungeeSpla）

Webアプリ･サイト

  • Salmon Learn - サーモンラーン



    いろいろ便利な鮭お役立ちWebアプリです。

  • 鮭ブキランキングメーカー



    鮭ブキランキングを作ることができます。

  • STタイマー



    サーモンラーンから独立したSTタイマーです。

  • ST+1分タイマー



    ST+1分タイマー。

  • ST+N分タイマーへのリンク生成ツール



    任意のST+N分タイマーへのリンクを生成できます。

  • YouTube2窓実況支援ツール



    YouTubeの野良2配信などを2窓実況するためのツールです。

  • Discord Overlay カスタムCSS参考ページ



    OBSにDiscordの通話アイコンを表示するときのカスタムCSS例を載せています。

ツール

  • HTML5エフェクトジェネレーター Ver.0.0.3



    コメントに反応してOBSの配信画面に自動で画像や動画エフェクトを表示することができるツールです。

素材

  • バイト服のイカちゃんのLive2Dモデル



    バイト服のイカちゃんがLive2Dで動くぞ！

  • オリマーBB



    『ピクミン』に登場するキャプテン・オリマーのBB素材。
